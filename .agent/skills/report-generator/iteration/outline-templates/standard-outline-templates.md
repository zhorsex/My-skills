# 标准大纲模板库

## 使用说明

本文档包含标准报告的大纲模板，用于快速生成符合行业规范的报告结构。

## 模板结构

每个模板包含：
- **模板ID**：唯一标识
- **报告类型**：报告分类
- **适用场景**：适用场景说明
- **详细程度**：精简/标准/详细
- **章节数量**：章节数量范围
- **标准结构**：章节标准结构（章+节）
- **写作要点**：写作注意事项

## 使用指南

### 模板选择建议

| 报告类型 | 推荐模板 | 理由 |
|---------|---------|-----|
| 内部快速汇报 | OT001（精简版） | 简洁高效，突出重点 |
| 可行性研究报告 | OT002（标准版） | 内容全面，结构完整 |
| 科研技术报告 | OT003（详细版） | 深度分析，论证充分 |
| 技术设计评审 | OT004（技术方案） | 技术准确，参数清晰 |
| 项目施工组织 | OT005（实施方案） | 可操作性强，计划具体 |

### 模板组合使用

不同模板可以根据实际情况进行组合，例如：
- 从 OT002（标准版）选取部分章节
- 从 OT003（详细版）补充深度分析
- 从 OT004（技术方案）引用技术内容
