# 环境评估报告大纲模板

## 基本信息
- 模板ID: OT103
- 报告类型: 环境影响评估报告
- 适用场景: 新建项目、改扩建项目、专项环评
- 详细程度: 中等（章+节）
- 行业标准: HJ 169-2018、HJ 2.1-2016、HJ 2.4-2009

## 写作要点
- 符合HJ 169-2018建设项目环境影响报告表编制技术导则
- 现状调查资料充分、监测数据可靠
- 影响预测方法合理、结论可信
- 保护措施具体可行、经济合理
- 公众参与过程规范、意见处理得当
