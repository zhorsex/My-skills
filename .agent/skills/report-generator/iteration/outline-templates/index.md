## 模板快速选择

### 根据报告类型选择

**问**: 您的报告类型是什么？

**选项**:
- A. 快速汇报/总结 → 推荐使用 **OT001**
- B. 可行性研究报告 → 推荐使用 **OT002**
- C. 技术白皮书/详细设计 → 推荐使用 **OT003**
- D. 技术设计方案 → 推荐使用 **OT004**
- E. 项目实施方案 → 推荐使用 **OT005**
- F. 地质调查报告 → 推荐使用 **OT101**
- G. 工程设计方案 → 推荐使用 **OT102**
- H. 环境影响评估 → 推荐使用 **OT103**

- C. 7-9章（详细）→ 推荐使用 **OT003, OT101, OT102, OT103**

## 使用说明

### 基础使用
1. 根据上述快速选择指南，选择合适的模板
2. 点击模板链接查看详细内容
3. 将模板内容复制到 report-generator 的大纲生成流程
4. 根据实际需要调整和定制

### 高级使用
1. 可以组合多个模板的内容
2. 可以根据项目特点定制模板
3. 使用统计信息了解常用模板
4. 根据满意度反馈选择最佳模板
