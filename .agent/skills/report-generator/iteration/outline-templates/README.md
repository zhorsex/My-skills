# 大纲模板库

本目录包含 report-generator 交互式大纲生成功能使用的模板文件。

## 目录结构

- `standard-outline-templates.md` - 标准大纲模板（精简版/标准版/详细版）
- `industry-outlines/` - 行业大纲模板（地质调查/工程设计/环境评估等）
- `index.md` - 模板索引

## 模板格式

每个模板包含：
- 模板ID
- 报告类型
- 适用场景
- 标准结构（章+节）
- 写作要点

## 使用方法

在 report-generator 生成大纲时，系统会自动从本目录检索相关模板。用户也可以指定使用特定模板。
