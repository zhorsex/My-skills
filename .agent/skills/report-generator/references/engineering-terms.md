## 术语使用规范

### 1. 首次使用
- 首次出现时给出简明定义
- 定义应简洁明了，避免冗长
- 对于常用术语，可使用脚注或括号说明

### 2. 后续使用
- 使用统一术语，避免同义词混用
- 可使用缩写（需在首次使用时注明）
- 保持术语的一致性

### 3. 避免歧义
- 对于有多种解释的术语，明确其在报告中的具体含义
- 使用行业公认的标准定义
- 必要时提供参考文献

### 4. 中英文对照
- 必要时可提供英文术语对照
- 英文术语使用斜体
- 在首次出现时同时给出中英文

## 术语一致性检查

### 检查要点
- [ ] 同一术语全文使用统一表述
- [ ] 术语定义与首次使用一致
- [ ] 缩写使用规范，首次使用已注明
- [ ] 无歧义术语或已明确含义
- [ ] 中英文对照使用正确

### 冲突处理
1. 发现术语使用不一致时：
   - 列出冲突项
   - 确认统一方式
   - 批量替换
   - 更新术语表

2. 发现术语定义冲突时：
   - 查阅权威资料
   - 确认行业标准定义
   - 在报告中明确采用的定义
